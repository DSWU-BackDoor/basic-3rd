#week04 과제
진행 과정:
1. open browser를 눌러 브라우저를 열어 덕성여자대학교 홈페이지에 접속한다.
2. Intercept on 을 누르고 홈페이지를 새로 고침한다
3. 요청 패킷을 foward한다
4. 응답 패킷을 클릭 후 키워드 검색창에 '웹메일'을 검색한다.
5. 웹메일 텍스트를 삭제한 후 '규리짱'을 입력한다
6. 변조된 응답 패킷을 foward한 후 intercept off를 누른다.
7. 브라우저로 이돔하여 변경사항을 확인한다.
