3. 덕성여대 검색

<img width="2879" height="1698" alt="Image" src="https://github.com/user-attachments/assets/b6fa4091-2e0c-43e9-bc72-a47cb4e27486" />

Forward all로 변경하고 브라우저가 열리면 검색창에 덕성여대를 검색하여 홈페이지로 접근한다. Forward했기 때문에 응답 패킷이 잡힌 것을 확인할 수 있다.
